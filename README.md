💬 Random Quote Generator

A modern and interactive Random Quote Generator web application built using HTML, CSS, and JavaScript. This app displays inspiring quotes along with their authors, helping users discover new ideas and motivation with every click or page refresh.

The application features a visually appealing quote display area with smooth animations that enhance the experience when switching between quotes. Users can generate new quotes instantly, copy them to the clipboard, or share them directly on social platforms like Twitter/X.

Designed with a clean and modern interface, the app includes light and dark theme switching, responsive layouts for all devices, and dynamic UI elements that create an engaging and polished user experience.

Core Features:

- Display a random quote and author on page load
- “Generate Quote” button for new quotes
- Large, visually appealing quote display section
- Smooth animations and transitions between quotes
- Quotes stored locally (array) or fetched from a public API
- Copy quote to clipboard functionality
- Social sharing option (e.g., Twitter/X)
- Light/Dark mode toggle
- Fully responsive design (mobile, tablet, desktop)

UI/UX Highlights:

- Clean and modern interface
- Elegant typography for readability
- Smooth transitions and subtle animations
- User-friendly layout and controls

Advanced Features (Optional):

- Auto-generate quotes at set intervals (toggle on/off)
- Category-based filtering (motivation, success, life, etc.)
- Dynamic background color or gradient changes
- Loading animation when fetching quotes from an API

This project is well-structured, beginner-friendly, and demonstrates key JavaScript concepts such as DOM manipulation, event handling, and working with APIs, while maintaining strong focus on performance and user experience.
